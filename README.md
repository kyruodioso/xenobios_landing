# Xenobios: El Despertar

**Xenobios** es un MMO RPG táctico y de supervivencia multiplataforma. Los jugadores asumen el rol de "Ánimas" en los Jardines del Génesis, vinculándose con entidades de energía llamadas "Mofis" en un ecosistema donde el riesgo y la recompensa definen tu nivel.

## 🛠️ Stack Tecnológico (Core)
- **Frontend & UI:** Next.js (App Router), React, Tailwind CSS, Framer Motion.
- **Motor 3D (In-Engine):** Three.js, React Three Fiber (R3F), Drei.
- **Backend & Auth:** Supabase (PostgreSQL, Edge Functions, Row Level Security).
- **Estado Global:** Zustand.
